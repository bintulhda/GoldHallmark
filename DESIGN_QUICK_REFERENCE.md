hardcode color/spacing values\n2. **Maintain hierarchy** - Use proper heading tags (h1-h6)\n3. **Respect spacing** - Use spacing variables for consistency\n4. **Add transitions** - Use `var(--transition-base)` for smooth interactions\n5. **Test on mobile** - Ensure responsive behavior works well\n6. **Test contrast** - Verify text color contrast meets WCAG AA+\n7. **Document custom styles** - Add comments for non-standard CSS\n8. **Reuse components** - Build modular, reusable CSS\n\n## File Structure\n\n```\nsrc/styles/\n├── theme.css          # Design system & variables\n├──
