# Premium Gold Theme - Visual Component Guide\n\n## Component Gallery & Usage\n\n### 1. BUTTONS\n\n#### Primary Button (Gold Outlined)\n```html\n<button class=\"btn btn-primary\">Click Me</button>\n```\n- Border: 2px solid gold\n- Background: Transparent\n- Text Color: Gold\n- Hover: Filled with gold, glow effect\n- Transition: 0.3s smooth\n\n#### Secondary Button (Light Gray)\n```html\n<button class=\"btn btn-secondary\">Secondary</button>\n```\n- Border: 2px solid light gray\n- Text: Light gray\n- Hover: Filled with light gray\n\n#### Success Button (Green)\n```html\n<button class=\"btn btn-success\">Success</button>\n```\n- Gold border, green filled on hover\n- Used for confirmations\n\n#### Size Variants\n```html\n<button class=\"btn btn-primary btn-sm\">Small</button>\n<button class=\"btn btn-primary\">Normal (default)</button>\n<button class=\"btn btn-primary btn-lg\">Large</button>\n```\n\n---\n\n### 2. CARDS\n\n#### Basic Card\n```html\n<div class=\"card\">\n  <div class=\"card-header\">\n    <h3>Title</h3>\n  </div>\n  <div class=\"card-body\">\n    <p>Content goes here</p>\n  </div>\n  <div class=\"card-footer\">\n    <p>Footer text</p>\n  </div>\n</div>\n```\n- Background: Dark (rgba with opacity)\n- Border: 1px subtle gold\n- Shadow: Soft + gold glow\n- Hover: Elevated with enhanced glow\n- Rounded: 16px\n\n#### Premium Feature Card\n```html\n<div class=\"card\">\n  <div style=\"font-size: 3rem; margin-bottom: 1rem;\">🏆</div>\n  <h3>Feature Title</h3>\n  <p>Description text</p>\n  <a href=\"#\" class=\"feature-link\">Learn more →</a>\n</div>\n```\n\n---\n\n### 3.
